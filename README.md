# Twinkle_25SCS1003002610_IILM-GN
AI Teaching Nexus is an advanced educational interface designed to help students master complex technical concepts through Multi-Agent Simulation. Instead of a standard chatbot, this platform uses specialized AI personas to provide theoretical, practical, and simplified perspectives simultaneously.

🚀 Core Features
🎭 Multi-Agent Debate Mode: Engage with three distinct AI experts:

Dr. Theory: Focuses on mathematical foundations and academic rigor.

Prof. Practice: Provides real-world code snippets and industry use cases.

Ms. Simple: Breaks down jargon using intuitive analogies.

🖼️ Dynamic SVG Visualizer: Integrated engine that generates live, labeled architectural diagrams and flowcharts to explain concepts spatially.

📈 Adaptive Complexity: Customizable learning paths for Beginner, Intermediate, and Advanced levels.

💎 Glassmorphic UI: A premium, high-performance interface built with Tailwind CSS and Lucide Icons for an immersive study experience.

🛠️ Technical Architecture
Frontend: React.js (Hooks, Functional Components)

Styling: Modern CSS-in-JS via Tailwind utility classes.

Intelligence: Anthropic Claude API integration for high-reasoning pedagogical responses.

Visuals: Real-time SVG rendering based on LLM-generated code.
